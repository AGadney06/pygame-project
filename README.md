# pygame-project

Requirements:
1. A full invaders array with 3 alien types ✅
2. The aliens should move from side to side ✅
3. Their movement should increase as the invaders are reduced in number (from being shot)
4. The defender player should be able to shoot continuously ✅ 
5. Implement collision detection so that invaders are destroyed when successfully shot by the player ✅
6. The player loses a life and respawns (until run out of lives) when shot by an invader
7. The four green barriers should diminish each time they are shot by both the defender and the invaders ✅
